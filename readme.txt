== Description ==

WordPress related posts will show related posts below article in very nice format. Plugin supports different templates for related posts.
It will help you to increase time per visit by showing to user related similar articles. Plugin is fully configurable and very easy to use.

Plugin features:

* Simple form for quick and easy configuration.
* Shortcode generator (you can use shortcodes to show related posts anywhere you want)
* Different templates (boxes and list)
* Plugin will use featured image, or first image from article
* Quick and optimized code for fast page load

Demos:

* [Example of page with related posts](http://demo.wp-resources.com/related-post-testing-article/)


== Installation ==

1. Unzip `wordpress-related-posts.zip` to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the configuration page in left sidebar
4. (Optional) Place shortcode anywhere you want
